Upload images for object detection. Marvin will identify and label objects with confidence percentages.

About Marvin AI

Marvin is an intelligent virtual assistant designed to demonstrate the capabilities of modern web technologies in creating powerful, browser-based AI applications.

The project was developed as an off-campus initiative to explore the integration of machine learning models directly in the browser, eliminating the need for server-side processing while maintaining high performance.

Our main goal with Marvin is to create an accessible, privacy-focused AI assistant that can perform complex tasks like object detection, voice recognition, and natural language processing entirely on the client side.

The motivation behind building Marvin stems from the desire to push the boundaries of what's possible with modern web technologies. By leveraging TensorFlow.js and Web APIs, we've created an assistant that works across devices without installations or complex setups.

Marvin represents the future of edge computing in web applications, where powerful AI capabilities are brought directly to users' browsers, ensuring privacy, reducing latency, and providing seamless experiences.
